upsetjs_jupyter_widget¶

Version: 0.6.0

A Jupyter Widget Library around UpSet.js

Quickstart¶

To get started with upsetjs_jupyter_widget, install with pip:

pip install upsetjs_jupyter_widget

or with conda:

conda install upsetjs_jupyter_widget
